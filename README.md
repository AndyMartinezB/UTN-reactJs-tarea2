# Tarea Nº2: Formulario interactivo con validación básica

## Descripción del Proyecto

Este proyecto corresponde a la segunda entrega del Curso de Desarrollo en React JS de
la UTN.BA. La tarea consiste en practicar la declaración de variables (let, const), el uso de operadores,
funciones, DOM y eventos en un pequeño formulario que valide datos y muestre un
mensaje dinámicamente.

## Instrucciones

Para clonar el repositorio en tu terminal:

```bash
git clone https://github.com/AndyMartinezB/UTN-reactJs-tarea2.git
```

Para ver la página:

- Abre el archivo `index.html` en tu navegador
- O si usas VS Code, podés utilizar la extensión Live Server.
